## CMS for blog
[![Build Status](https://travis-ci.org/wildtype/ochre.png?branch=master)](https://travis-ci.org/wildtype/ochre)

Install dependencies:

```
$ yarn
```

Build in development mode:

```
$ yarn production-build
```

Run the tests:

```
$ yarn test
```

To build with uglified js:

```
$ yarn
$ yarn production-build
